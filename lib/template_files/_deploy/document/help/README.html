团800 Wiki 系统

基本命令

  1. 更新代码

  2. 安装GEMS:

    bundle install

  3. 开发环境开启服务器

  • bundle exec docwu s # 默认开启， http://127.0.0.1:5656
  • bundle exec docwu s -p 3300 # 以3300端口开启服务器， http://127.0.0.1:3300
  1. 当文档修改了，需要使用如下命令进行重新生成
  • bundle exec docwu g
